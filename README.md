# Портфолио - Мирзомансур Охунов

Персональное портфолио продуктового дизайнера, сверстанное на основе дизайна из Figma.

## Структура проекта

- `index.html` - основная HTML-разметка
- `styles.css` - стили страницы
- `assets/` - папка для изображений

## Использование

1. Откройте `index.html` в браузере
2. Добавьте изображения в папку `assets/`:
   - `main-image.jpg` - основное изображение (224x224px)
   - `case-1.jpg` - изображение для кейса "Кредитная история" (518x532px)
   - `case-2.jpg` - изображение для кейса "Концепт прематча" (518x532px)
   - `case-3.jpg` - изображение для кейса "Дизайн система" (518x532px)
   - `case-4.jpg` - изображение для кейса "Фитнес приложение" (518x532px)

## Особенности

- Адаптивный дизайн для мобильных устройств
- Чистый HTML/CSS/JS без фреймворков
- Соответствие дизайну из Figma
